Perception with **dynamic vision sensors (event cameras)**: Kalman-filter-based
tracking pipelines, spiking neural network / R-STDP experiments, and a ROS2
package for handling DVS event streams. Event cameras offer microsecond latency
and high dynamic range — useful where conventional frames blur or saturate.
